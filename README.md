# monkey-rs
An implementation of the
[Monkey](https://interpreterbook.com/#the-monkey-programming-language)
programming language in Rust. This is based on
the [Writing an Interpreter in Go](https://interpreterbook.com/) book by
Thorsten Ball.

## Extra Features
Here are a few extra features in this implementation of Monkey:
- Alphabetic unicode character support for identifiers ("ä¸€", for example)
- Numbers in identifiers where the number is not the first part of the
  identifier ("x1", for example)
